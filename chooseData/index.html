日期选择器 #

用于选择开始/结束类的数据（例如起止日期），组件通过事件回传开始与结束值，便于在表单或筛选中使用。

主要 API：

  • 事件

    • startChange：开始值改变时触发，参数通常为 { startDate, endDate } 或开始值。
    • endChange：结束值改变时触发。

  • Props

    • startOptions / endOptions：可传入配置对象，透传给内部控件（例如尺寸、占位等）。

示例用法参考 src/views/chooseData/index.vue：

<chooseData
 @startChange="startChange"
 @endChange="endChange"
 :startOptions="startOptions"
 :endOptions="endOptions"
></chooseData>
