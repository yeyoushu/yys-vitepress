趋势图组件 #

用于展示小型趋势图或图表摘要，适合在列表或卡片中快速展示数据走势。

按需引入 #


import chooseTrend from 'yys-element-components/trend'
import 'yys-element-components/trend/style.css'
// min.ts中注册
app.use(chooseTrend)
