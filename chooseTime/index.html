时间选择器 #

用于选择开始/结束时间，组件通过 startChange / endChange 事件回传值，适合筛选或表单中的时间段选择。

按需引入 #


import chooseTime from 'yys-element-components/chooseTime'
import 'yys-element-components/chooseTime/style.css'
// min.ts中注册
app.use(chooseTime)

主要 API：

  • Props

    • startOptions / endOptions：传入配置对象，控制内部控件行为（例如尺寸、占位等）。

  • 事件

    • startChange：开始时间改变时触发，参数为字符串或自定义结构。
    • endChange：结束时间改变时触发。

代码示例 #

<chooseTime
 :startOptions="startOptions"
 :endOptions="endOptions"
 @startChange="startChange"
 @endChange="endChange"
></chooseTime>

<script setup lang="ts">
interface endValue {
  startTime: string;
  endTime: string;
}
let startChange = (val: string) => {
  console.log("开始时间", val);
};
let endChange = (val: endValue) => {
  console.log("结束时间", val);
};
let startOptions = {
  //   size: "small",
};
let endOptions = {};
</script>
