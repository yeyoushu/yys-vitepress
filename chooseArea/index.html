区域选择器 #

用于省/市/区三级选择，支持传入完整数据、开启搜索过滤、定制占位文案、尺寸和清除功能；当选择到区时会触发 change 事件并返回选中结果对象。

按需引入 #

import chooseArea from 'yys-element-components/chooseArea'
import 'yys-element-components/chooseArea/style.css'
// 在全局注册：
app.use(chooseArea)

组件要点 #

  • Props

    • allAreasData：完整的省市区数据（数组，结构见示例）。
    • filterable：是否启用输入搜索（默认 false）。
    • filterMethod：自定义过滤函数 (query, item) => boolean。
    • placeholders：占位文案数组，如 ['请选择省份','请选择城市','请选择区域']。
    • size：尺寸，支持 '' | 'default' | 'small' | 'large'。
    • clearable：是否显示清除按钮。

  • 事件

    • change：当用户选择到区（area）时触发，回调参数为 { province: {code,name}, city: {code,name}, area: {code,name} }。

  • API

    • 组件通过 defineExpose 暴露 reset() 方法，用于重置已选值（可通过 ref 调用）。

默认用法：

<chooseArea @change="onChangeDefault" :allAreasData="allAreasData" />

高级用法（启用搜索、定制占位、尺寸、清除与重置）：

<chooseArea
     ref="areaRef"
     :allAreasData="allAreasData"
     :filterable="true"
     :filterMethod="filterByName"
     :placeholders="[
          '请选择省(带搜索)',
          '请选择市(带搜索)',
          '请选择区(带搜索)'
     ]"
     size="large"
     :clearable="true"
     @change="onChangeAdvanced"
/>

<el-button @click="resetAdvanced">重置（调用组件 reset）</el-button>

import allAreas from 'path/to/pca-code.json' // 替换为你的省市区数据文件
import { ref } from 'vue'

const allAreasData = ref(allAreas)
const defaultResult = ref(null)
const advancedResult = ref(null)

function onChangeDefault(val){ defaultResult.value = val }
function onChangeAdvanced(val){ advancedResult.value = val }

function filterByName(query, item){
     if(!query) return true
     return String(item.name || '').toLowerCase().includes(String(query).toLowerCase())
}

const areaRef = ref(null)
function resetAdvanced(){ areaRef.value?.reset?.() }
