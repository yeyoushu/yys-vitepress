通知组件 #

显示通知菜单列表，支持多种自定义。

按需引入 #


import chooseList from 'yys-element-components/list'
import 'yys-element-components/notification/style.css'
// min.ts中注册
app.use(chooseList)

基本用法 #


示例 #

  <div class="notification-demo">
    <chooseList
      :list="list"
      :actions="actions"
      @clickItem="clickItem"
      @clickAction="clickAction"
    >
    </chooseList>
  </div>

API #

事件list #

属性说明类型可选值默认值
list列表内容ListOptions[]必填-
actions操作内容ActionOptions[]--

ListOptions 数据格式 #

key说明类型
titletab的标题string
content列表的类listItem[]

listItem 数据格式 #

key说明类型
title头像string
avatar标题string
desc描述string
time时间string
tag标签内容string
tagType列表的类'' / success / info / warning / danger

ActionOptions 数据格式 #

key说明类型
text操作的标题string
icon操作的图标string
通知菜单
导航菜单
