快速上手 #

当前组件库是基于element-plus二次封装的，需确保项目中已经安装引入正确的element-plus。element-plus安装使用指南

使用指南 #

1.在项目中安装 yys-element-components 推荐：npm i -S yys-element-components cnpm安装：cnpm i -S yys-element-components 2.再main.ts文件中写入以下代码。 全局引入

import mUi from 'yys-element-components'
import 'yys-element-components/style.css'
import { createApp } from 'vue'
import ElementPlus from 'element-plus'
import 'element-plus/dist/index.css'
import App from './App.vue'
// 按需引入引入方式
import chooseIcon from 'yys-element-components/chooseIcon'
// 引入对应的css
import  from 'yys-element-components/chooseIcon/style.css'
import chooseArea from 'yys-element-components/chooseArea'
import chooseTrend from 'yys-element-components/trend'
import chooseList from 'yys-element-components/list'
import notification from 'yys-element-components/notification'
import chooseMenu from 'yys-element-components/menu'
import chooseProgress from 'yys-element-components/progress'
import chooseTime from 'yys-element-components/chooseTime'
import chooseData from 'yys-element-components/chooseData'
import chooseCity from 'yys-element-components/chooseCity'
import chooseFrom from 'yys-element-components/from'
import modalFrom from 'yys-element-components/modalFrom'
import chooseTable from 'yys-element-components/table'
import chooseCalendar from 'yys-element-components/calendar'
const app=createApp
app.use(ElementPlus)
// 示例
app.use(chooseIcon)
app.mount('#app')

4.使用 在需要的组件内参照组件文档了解具体使用的方式
