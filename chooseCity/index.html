城市选择器 #

支持中国省市区选择所有城市不含港澳台（chooseStandbyCity / chooseCity）。

按需引入 #


import chooseCity from 'yys-element-components/chooseCity'
import chooseStandbyCity from 'yys-element-components/chooseCity'
import 'yys-element-components/chooseCity/style.css'
// min.ts中注册
app.use(chooseCity)
app.use(chooseStandbyCity)

基本用法 #


选中结果：尚未选择

代码示例 #

    <div style="display: flex; gap: 12px">
      <chooseStandbyCity @change="onChange" />
      <chooseCity
        @change="onChange"
        @changeCity="onChange"
        @changeProvince="onChange"
      />
    </div>
    <div style="margin-top: 12px">选中结果：{{ result }}</div>
  </div>

用于选择省/市/区，包含备用（standby）和主控两种组件，用法示例见下。

按需引入代码示例 #

基本用法 #

<template>
 <div style="display: flex">
  <chooseStandbyCity @change="chooseChange"></chooseStandbyCity>
  <chooseCity></chooseCity>
 </div>
</template>

<script setup lang="ts">
import { ref } from "vue";
const result = ref("尚未选择");
function onChange(val: any) {
  result.value = JSON.stringify(val);
}
</script>

属性说明类型

chooseStandbyCity #

属性说明类型
change选择城市或省份时触发val: 城市信息
  • change：选中后回调选中结果对象。
